# Implement DHCPv4 Lab
This lab demonstrates the configuration of a centralized DHCPv4 server on a Cisco Router.

## Objectives
* Configured **R2** as a DHCP Server for multiple LANs.
* Implemented **DHCP Relay (ip helper-address)** on R1 and R3.
* Configured **R2** as a DHCP Client to receive an IP from an ISP.
